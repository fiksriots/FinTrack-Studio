Panduan Penggunaan FinTrack Studio

1. Cara Install di HP Android (PWA)

Buka web ini di Google Chrome HP Android → klik tombol Install App Android di sidebar (atau klik titik 3 Chrome → pilih Add to Home Screen). Aplikasi akan terpasang di HP Anda & bisa digunakan 100% offline!

2. Cara Backup & Sync Google Drive

Buka menu Database & Backup → bagian 3. Sync Cloud → klik Hubungkan Google. Setelah terhubung, klik Backup ke Google Drive untuk menyimpan cadangan ke cloud pribadi Anda secara aman.

3. Cetak Laporan Keuangan PDF

Buka menu Transaksi atau Database & Backup → klik Cetak Laporan PDF. Anda bisa memilih periode (Bulan Ini, Tahun Ini, dll) & mencetak dokumen laporan eksekutif resmi.

4. Lampiran Struk & Fitur Lainnya

Saat menambah transaksi, Anda bisa melampirkan foto nota/struk belanjaan. Gunakan juga tombol Privasi di header untuk mengaburkan saldo saat berada di tempat umum.
